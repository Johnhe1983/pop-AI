# pop-AI 🤖
AI 运营助理军团 | 千川 / 本地推 / 聚光 / 直播间 / 短视频 / POI 自动化工具集

专为 AI OPC（一人公司）打造的全流程运营工具，无需团队，一人掌控全链路投放与运营。

---

## 🎯 核心功能模块
### 1. 广告投放自动化
- 巨量千川计划创建、文案生成、数据复盘
- 抖音本地推团购投放、POI 引流脚本
- 小红书聚光平台创意、标题、投放策略
- 自动生成高转化投放文案

### 2. 直播间运营
- 直播全流程脚本生成
- 逼单话术、互动话术、过品话术
- 直播数据复盘与优化建议
- 自动切片高光片段

### 3. 短视频运营
- 短视频爆款脚本（前3秒钩子）
- AI 文案、字幕、配音
- 图文成片、数字人口播
- 批量生成短视频内容

### 4. POI 门店运营
- POI 创建与信息优化
- 门店介绍、团购套餐文案
- 门店封面图、环境图、套餐图描述
- 本地生活引流话术
